# SkPaint Basics

`SkPaint` controls how geometry is drawn. The canvas decides **where** drawing happens. `SkPaint` decides **how** it looks.

Typical paint properties include:

- fill or stroke style
- color and alpha
- stroke width
- anti-aliasing
- effects such as path effects, shaders, and color filters

It is also used in some non-drawing APIs to control measurement and styling behavior.

## Just a Paint

```cpp
canvas->clear(SK_ColorWHITE);
SkPaint paint;
canvas->drawRoundRect({ 40,40,200,300 }, 10, 10, paint);
```

## setPathEffect

Example 1, dashed effect:

```cpp
canvas->clear(SK_ColorWHITE);
SkPaint paint;
paint.setColor(SK_ColorRED);
paint.setStyle(SkPaint::kStroke_Style);
paint.setStrokeWidth(5);
paint.setAntiAlias(true);
paint.setAlpha(255);
SkScalar intervals[] = { 20, 14 };
paint.setPathEffect(SkDashPathEffect::Make(intervals, 5));
canvas->drawRoundRect({ 40,40,200,300 }, 10, 10, paint);
```

![SkDashPathEffect example](/img/image24.png)

Example 2, trim path effect:

```cpp
canvas->clear(SK_ColorWHITE);
SkPaint paint;
paint.setColor(SK_ColorRED);
paint.setStyle(SkPaint::kStroke_Style);
paint.setStrokeWidth(5);
paint.setAntiAlias(true);
paint.setAlpha(255);
paint.setPathEffect(SkTrimPathEffect::Make(0.25f, 0.75f));
canvas->drawRoundRect({ 40,40,200,300 }, 10, 10, paint);
```

![SkTrimPathEffect example](/img/image25.png)

Example 3, composed path effects:

```cpp
canvas->clear(SK_ColorWHITE);
SkPaint paint;
paint.setColor(SK_ColorRED);
paint.setStyle(SkPaint::kStroke_Style);
paint.setStrokeWidth(5);
paint.setAntiAlias(true);
paint.setAlpha(255);
SkScalar intervals[] = { 20, 14 };
auto trim = SkTrimPathEffect::Make(0.25f, 0.75f);
auto dash = SkDashPathEffect::Make(intervals, 5);
paint.setPathEffect(SkPathEffect::MakeCompose(
    dash,
    trim
));
canvas->drawRoundRect({ 40,40,200,300 }, 10, 10, paint);
```

## setShader

```cpp
canvas->clear(SK_ColorWHITE);
SkPaint paint;
std::array<SkColor4f, 3> colors = {
    SkColors::kRed,
    SkColors::kYellow,
    SkColors::kBlue
};
std::array<float, 3> pos = { 0.0f, 0.5f, 1.0f };

SkGradient grad = {
    { colors, pos, SkTileMode::kClamp },
    {}
};

SkPoint pts[2] = { {0, 0}, {300, 0} };
paint.setShader(SkShaders::LinearGradient(pts, grad));
paint.setStyle(SkPaint::kStroke_Style);
paint.setStrokeWidth(5);
paint.setAntiAlias(true);
paint.setAlpha(255);
SkScalar intervals[] = { 20, 14 };
auto trim = SkTrimPathEffect::Make(0.25f, 0.75f);
auto dash = SkDashPathEffect::Make(intervals, 5);
paint.setPathEffect(SkPathEffect::MakeSum(
    dash,
    trim
));
paint.setStrokeCap(SkPaint::kRound_Cap);
canvas->drawRoundRect({ 40,40,200,300 }, 10, 10, paint);
```

![Shader example with gradient stroke](/img/image28.png)

## setColorFilter

```cpp
canvas->clear(SK_ColorWHITE);
std::filesystem::path path = "C:\\Users\\deb\\Downloads\\sunflower.jpg";
const std::string narrowPath = path.string();
sk_sp<SkData> data = SkData::MakeFromFileName(narrowPath.c_str());
sk_sp<SkImage> image = SkImages::DeferredFromEncodedData(data);
SkColorMatrix matrix;
matrix.setSaturation(0.0f);
SkPaint paint;
paint.setColorFilter(SkColorFilters::Matrix(matrix));
canvas->drawImage(image, 100, 100, SkSamplingOptions(), &paint);
```
