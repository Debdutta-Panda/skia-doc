# SkPaint Basics

`SkPaint` controls how geometry is drawn. The canvas decides **where** drawing happens. `SkPaint` decides **how** it looks.

Typical paint properties include:

- fill or stroke style
- color and alpha
- stroke width
- anti-aliasing
- effects such as path effects, shaders, and color filters

## Minimal example

```cpp
canvas->clear(SK_ColorWHITE);

SkPaint paint;
paint.setAntiAlias(true);
paint.setColor(SK_ColorRED);

canvas->drawRoundRect({40, 40, 200, 300}, 10, 10, paint);
```

## Most-used paint settings

### Fill vs stroke

```cpp
paint.setStyle(SkPaint::kStroke_Style);
paint.setStrokeWidth(5);
```

### Anti-aliasing

```cpp
paint.setAntiAlias(true);
```

This is usually a good default for UI drawing and vector shapes.

### Alpha

```cpp
paint.setAlpha(128);
```

Alpha affects transparency. Lower alpha makes the result more transparent.

## Why paint matters

The same geometry can look completely different depending on the paint:

- a rectangle can be filled
- the same rectangle can be stroked
- it can become dashed with a path effect
- it can be tinted with a color filter

That makes `SkPaint` one of the most important objects to understand early.
