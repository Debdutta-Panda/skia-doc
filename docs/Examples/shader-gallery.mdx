# Skia Shader Gallery Example

This Skia shader gallery example puts the most useful shader patterns on one page so you can compare them in a single scene.

It demonstrates:

- linear gradient
- radial gradient
- sweep gradient
- two-point conical gradient
- image shader
- blended shaders

## Why this is useful

Many shader examples are too isolated.

A gallery layout is better because it lets you compare:

- how each shader maps into geometry
- where gradients feel soft or sharp
- how local transforms change shader direction
- when image-backed shaders are more appropriate than gradients

## Example

```cpp
#include <filesystem>
#include "include/core/SkCanvas.h"
#include "include/core/SkData.h"
#include "include/core/SkImage.h"
#include "include/core/SkMatrix.h"
#include "include/core/SkPaint.h"
#include "include/core/SkRect.h"
#include "include/core/SkShader.h"
#include "include/effects/SkGradientShader.h"

static void DrawTile(
    SkCanvas* canvas,
    const char* label,
    const SkRect& rect,
    sk_sp<SkShader> shader,
    const SkFont& font
) {
    SkPaint fill;
    fill.setAntiAlias(true);
    fill.setShader(std::move(shader));

    SkPaint stroke;
    stroke.setAntiAlias(true);
    stroke.setStyle(SkPaint::kStroke_Style);
    stroke.setStrokeWidth(2);
    stroke.setColor(SkColorSetARGB(90, 30, 30, 30));

    SkPaint textPaint;
    textPaint.setAntiAlias(true);
    textPaint.setColor(SkColorSetRGB(35, 35, 35));

    canvas->drawRoundRect(rect, 20, 20, fill);
    canvas->drawRoundRect(rect, 20, 20, stroke);
    canvas->drawString(label, rect.left(), rect.bottom() + 26, font, textPaint);
}

void DrawScene(SkCanvas* canvas, const AppState& state) {
    canvas->clear(SkColorSetRGB(246, 243, 236));

    skiaexplorer::FontSystem fonts;
    fonts.Initialize();
    SkFont font(fonts.MatchTypeface("Segoe UI", SkFontStyle()), 18.0f);

    const SkRect a = SkRect::MakeXYWH(40, 50, 220, 140);
    const SkRect b = SkRect::MakeXYWH(300, 50, 220, 140);
    const SkRect c = SkRect::MakeXYWH(560, 50, 220, 140);
    const SkRect d = SkRect::MakeXYWH(40, 280, 220, 140);
    const SkRect e = SkRect::MakeXYWH(300, 280, 220, 140);
    const SkRect f = SkRect::MakeXYWH(560, 280, 220, 140);

    const SkColor colors[4] = {
        SkColorSetRGB(242, 92, 84),
        SkColorSetRGB(249, 190, 79),
        SkColorSetRGB(76, 178, 151),
        SkColorSetRGB(67, 107, 203)
    };
    const SkScalar pos[4] = { 0.0f, 0.33f, 0.66f, 1.0f };

    SkPoint linearPts[2] = { {a.left(), a.top()}, {a.right(), a.bottom()} };
    auto linear = SkGradientShader::MakeLinear(
        linearPts,
        colors,
        pos,
        4,
        SkTileMode::kClamp
    );

    auto radial = SkGradientShader::MakeRadial(
        b.center(),
        b.width() * 0.45f,
        colors,
        pos,
        4,
        SkTileMode::kClamp
    );

    auto sweep = SkGradientShader::MakeSweep(
        c.center(),
        colors,
        pos,
        4
    );

    auto conical = SkGradientShader::MakeTwoPointConical(
        SkPoint::Make(d.left() + 60, d.centerY()),
        10.0f,
        SkPoint::Make(d.right() - 50, d.centerY()),
        110.0f,
        colors,
        pos,
        4,
        SkTileMode::kClamp
    );

    std::filesystem::path imagePath = "C:\\Users\\deb\\Downloads\\small-cat.webp";
    sk_sp<SkData> data = SkData::MakeFromFileName(imagePath.string().c_str());
    sk_sp<SkImage> image = data ? SkImages::DeferredFromEncodedData(data) : nullptr;

    sk_sp<SkShader> imageShader;
    if (image) {
        SkMatrix local;
        local.setScaleTranslate(0.35f, 0.35f, e.left(), e.top());
        imageShader = image->makeShader(
            SkTileMode::kRepeat,
            SkTileMode::kRepeat,
            SkSamplingOptions(),
            &local
        );
    } else {
        imageShader = SkShaders::Color(SkColorSetRGB(210, 210, 210));
    }

    SkPoint blendPts[2] = {
        { f.left(), f.top() },
        { f.right(), f.top() }
    };
    SkColor blendAColors[2] = {
        SkColorSetARGB(255, 255, 80, 70),
        SkColorSetARGB(255, 255, 215, 90)
    };
    auto blendA = SkGradientShader::MakeLinear(
        blendPts,
        blendAColors,
        nullptr,
        2,
        SkTileMode::kClamp
    );

    SkColor blendBColors[2] = {
        SkColorSetARGB(255, 70, 120, 230),
        SkColorSetARGB(30, 70, 120, 230)
    };
    auto blendB = SkGradientShader::MakeRadial(
        f.center(),
        90.0f,
        blendBColors,
        nullptr,
        2,
        SkTileMode::kClamp
    );
    auto blended = SkShaders::Blend(SkBlendMode::kScreen, blendA, blendB);

    DrawTile(canvas, "Linear Gradient", a, linear, font);
    DrawTile(canvas, "Radial Gradient", b, radial, font);
    DrawTile(canvas, "Sweep Gradient", c, sweep, font);
    DrawTile(canvas, "Two-Point Conical", d, conical, font);
    DrawTile(canvas, "Image Shader", e, imageShader, font);
    DrawTile(canvas, "Blended Shaders", f, blended, font);
}
```

## What to notice

- linear gradients are easiest to control for directional lighting
- radial gradients are great for soft focus and spotlight effects
- sweep gradients feel rotational and UI-like
- conical gradients create more dramatic spatial distortion
- image shaders are useful for texture-backed fills
- blended shaders can create richer color interactions than one gradient alone
